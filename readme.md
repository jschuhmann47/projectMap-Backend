# Project Map Backend

## Installation

```bash
git clone https://github.com/jschuhmann47/projectMap-Backend.git
cd projectMap-Backend
npm install
```

Required env vars:

- MONGO_URI
- SECRET_KEY
- SENDGRID_API_KEY
- PORT

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test
```

## Lint

```bash
# eslint
npm run lint
```
